# Projects

This is where every student's project lives. One folder per person.

---

## How to Add Yours

**1. Copy the starter template into this folder:**

```bash
# From the repo root
cp -r starter-template projects/your-name
```

**2. Rename the folder** to your first name, last name, or handle. Keep it lowercase, no spaces:

```
projects/sofia/
projects/marcus/
projects/ab/
```

**3. Install dependencies and start the dev server:**

```bash
cd projects/your-name
npm install
npm run dev
# Open http://localhost:5173
```

**4. Build your app** inside that folder. It's yours — experiment freely.

**5. When you're ready**, commit your changes and open a pull request.

---

## Rules

- Only edit files inside **your own folder**
- Do not modify `starter-template/` or `docs/`
- Do not commit `node_modules/` — it's already excluded by `.gitignore`
